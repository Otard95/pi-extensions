# pi-extensions

## Scripts

| Command | Description |
|---|---|
| `npm run check` | TypeScript type check (errors only) |
| `npm run check:full` | Full diagnostics incl. suggestions (TS80007 etc.) |
| `npm run lint:fix` | Biome lint + auto-fix |
| `npm run test` | Run tests |

## Development dependencies

There's dev dependencies in `./package.json` that you need to keep up do date
with:
```bash
pi --version
```
